# 📊 Projeto 1: Análise do Poder de Compra do Salário Mínimo (1994 - 2026)

Bem-vindo ao meu portfólio de Análise de Dados! Este é o primeiro projeto prático, onde investigamos a verdadeira evolução do poder de compra do trabalhador brasileiro, removendo a "ilusão monetária" causada pela inflação.

## 🎯 O Desafio de Negócio
Muitas vezes, a mídia e os gráficos governamentais mostram apenas o crescimento do **Salário Nominal** (o número impresso na nota). O objetivo deste projeto foi usar o índice de inflação (IPCA/DIEESE) para calcular o **Salário Real**, provando matematicamente o quanto o padrão de vida do brasileiro realmente mudou desde a criação do Plano Real.

## 🛠️ Ferramentas Utilizadas
* **Linguagem:** Python
* **Manipulação de Dados:** Pandas
* **Visualização de Dados:** Matplotlib
* **Ambiente:** Jupyter Notebook (Ubuntu)

## 💡 Principais Insights (Data Storytelling)
Após o tratamento e análise dos dados, descobrimos que:
1. **A Ilusão Nominal:** O salário bruto subiu substancialmente em porcentagem de 1994 a 2026, porém, grande parte disso foi apenas correção inflacionária.
2. **O Ganho Real:** O crescimento *real* do poder de compra (o quanto a mais de comida se coloca no carrinho do supermercado) foi muito menor do que a curva nominal sugere.
3. **Pico e Queda:** O algoritmo identificou com precisão os anos em que o brasileiro teve o maior e o menor poder aquisitivo histórico, comprovando a volatilidade econômica do período.
